# Heart Disease Prediction AI Project

## Overview
This project predicts whether a patient is likely to have heart disease using machine learning techniques. It demonstrates the complete AI workflow from data preprocessing to deployment.

## Project Workflow
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Deployment using Streamlit

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Tuned Random Forest
- Support Vector Machine (SVM)

## Best Model
Decision Tree

**Accuracy:** 57.1%

**F1-Score:** 59.8%
